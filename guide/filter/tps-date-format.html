# 2. tpsDateFormat

格式化时间

2023-03-31 17:20:15

2022-05-20

05-23 10:11

<template>
  <div>
    <p class="txt-blue">{{ data1 | tpsDateFormat }}</p>
    <p class="txt-blue">{{ data2 | tpsDateFormat("yyyy-MM-dd") }}</p>
    <p class="txt-blue">{{ data3 | tpsDateFormat("MM-dd hh:mm") }}</p>
  </div>
</template>
<script>
export default {
  data() {
    return {
      data1: new Date(),
      data2: "2022-05-20 11:57:56",
      data3: 1653271889125,
    };
  },
};
</script>

tpsDateFormat可把字符串进行格式化。

该字符串需要符合要求：new Date(str)可成功转为时间对象。

tpsDateFormat的一个额外参数是格式化字符串，默认为yyyy-MM-dd hh:mm:ss。
